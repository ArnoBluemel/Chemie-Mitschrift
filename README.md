## Dev Info

### Software

- Git
- Node.js
- Code editor (Visual Studio Code)

### Setup

```
git pull https://github.com/ArnoBluemel/Chemie-Mitschrift.git

npm install
```

### Working on it

```
npm run dev
```

### Putting it live

```
npm run deploy
```
